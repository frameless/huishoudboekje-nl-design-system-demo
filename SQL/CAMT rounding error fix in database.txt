There was a rounding issue when using CAMT files to upload banktransactions. Use these sql statements to either check if it happend or update te wrong transactions.

To check if there are transactions that have a rounding error:

SELECT id, bedrag FROM bank_transactions WHERE ROUND(CAST(SUBSTRING(SUBSTRING(statement_line FROM 7),'([+-]?(?=\.\d|\d)(?:\d+)?(?:\.?\d*))(?:[Ee]([+-]?\d+))?')AS DECIMAL),2) *100 != bedrag;

To update transactions that have a rounding error:

UPDATE bank_transactions SET bedrag = ROUND(CAST(SUBSTRING(SUBSTRING(statement_line FROM 7),'([+-]?(?=\.\d|\d)(?:\d+)?(?:\.?\d*))(?:[Ee]([+-]?\d+))?')AS DECIMAL),2) * 100 WHERE ROUND(CAST(SUBSTRING(SUBSTRING(statement_line FROM 7),'([+-]?(?=\.\d|\d)(?:\d+)?(?:\.?\d*))(?:[Ee]([+-]?\d+))?')AS DECIMAL),2) *100 != bedrag;
