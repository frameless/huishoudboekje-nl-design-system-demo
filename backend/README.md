
# hhb_backend
Laag 4 in het Common Ground-lagenmodel. GraphQL-backendserver

## Instalatie
Normale installatie

```python3 setup.py install```

Development instalatie
```pip install -e .```
of
```pip install -r requirements.txt```

Dependencies moeten gedefineerd worden in het setup.py bestand.
