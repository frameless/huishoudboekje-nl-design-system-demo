# Section

You can use the Section-component to design a page. To have the white background with the rounded borders, wrap it in a SectionContainer:

	<SectionContainer>
		<Section>
			<Lipsum />
		</Section>
	</SectionContainer>

## Bare Section
<Canvas>
	<Story name="Bare">
		<SectionContainer>
			<Section>
				<Lipsum />
			</Section>
		</SectionContainer>
	</Story>
</Canvas>

## With options in the top bar
<Canvas>
	<Story name="With top bar">
		<SectionContainer>
			<Section
				title="Persoonlijk"
				helperText="Persoonlijke gegevens van de burger."
				menu={(
					<Menu>
						<IconButton as={MenuButton} icon={<ChevronDownIcon />} variant={"solid"} size={"sm"} aria-label={"Open menu"} />
						<MenuList>
							<MenuItem>Bekijk rapportage</MenuItem>
							<MenuItem>Wijzigen</MenuItem>
							<MenuItem>Verwijderen</MenuItem>
						</MenuList>
					</Menu>
				)}
				right={(
					<HStack>
						<Button colorScheme={"gray"} size={"sm"}>Wijzigen</Button>
						<Button colorScheme={"red"} variant={"outline"} size={"sm"}>Verwijderen</Button>
					</HStack>
				)}
			>
				<Lipsum />
			</Section>
		</SectionContainer>
	</Story>
</Canvas>
